# TaskIt - Task Manager

[![Laravel](https://img.shields.io/badge/Laravel%2012-%23FF2D20.svg?logo=laravel&logoColor=white)](https://laravel.com)
[![Angular](https://img.shields.io/badge/Angular%2021-%23DD0031.svg?logo=angular&logoColor=white)](https://angular.io)

Keep track of your tasks:
- add new tasks
- edit tasks
- search and filters your tasks
- view tasks by categories
- complete tasks and view the completed ones
- archive no more useful tasks

Full-stack task manager with Laravel 12 API backend and Angular 21 frontend. Features auth, CRUD, drag-and-drop reordering, and real-time search.
